# 🔐 KeyMaster Python Web Integration

A lightweight, secure, and fully working **Python + Flask-based login system** for authenticating users via the [KeyMaster AGNI](https://keymaster-agni.vercel.app) API.

This repo demonstrates:
- 🔑 License Key and Username/Password authentication
- 💻 Persistent HWID generation per user
- ⚡ Beautiful and animated HTML/CSS frontend
- 🔁 Easy-to-customize Flask backend

---
